雅虎14条web优化原则


雅虎14条优化原则
雅虎卓越的性能研究团队确定了许多使得web页面更快运行的的最佳实践。该列表包括35个最佳实践，分为7个类别。

The Exceptional Performance team has identified a number of best practices for making web pages fast. The list includes 35 best practices divided into 7 categories

减少HTTP请求(Minimize HTTP Requests) - content

使用CDN(内容分发网络: Content Delivery Network) - server

使用Expires,Cache-Control缓存(Add an Expires or a Cache-Control Header) - server

压缩组件，stylesheet, script (Gzip Components) - server

将CSS样式放在头部(Put Stylesheets at the Top) - css

将js放在底部(Put Scripts at the Bottom) - javascript

避免使用Expressions(Avoid CSS Expressions) - css

CSS和JS都外部引用(Make JavaScript and CSS External) - css, javascript

减少DNS查询(Reduce DNS Lookups) - content

减少Javascript和CSS （Minify JavaScript and CSS）- css, javascript

避免重定向(Avoid Redirects) - content

删除重复的JavaScript（Remove Duplicate Scripts）- javascript

使用实体标签(Configure ETags)
