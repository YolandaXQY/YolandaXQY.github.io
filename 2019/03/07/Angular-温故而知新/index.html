Angular-cli (Angular command-line interface)

  1. npm intall -g @angular-cli@latest
  2. 装yarn, npm ,cnpm, node;
  3. 创建模块: ng g module home –module=app –routing=true
  4. 创建组件: ng g component header -m=home

    yarn, cnpm 可能比npm 好用，因为npm的服务器在国外

基础学习

1. JQ是 事件驱动,Angular是 数据驱动

2. 插值表达式

即用双花括号即可引用TS中的变量,对象,数组内容等；插值表达式不仅能够用在纯文本的地方,还能用在属性上.

3. Angular绑定机制

Angular最大的亮点就是他的绑定机制,一共有三种

属性 - 从数据源到视图(单向)
事件 - 从视图到数据源(单向)
表单值 - 从视图到数据源再到视图(双向)

视图,就是我们看到的样子,其实就是指HTML.
数据源,是指typescript中的值,无论他是定义好的值还是来自服务器的值,都是数据源.
