JavaScript循环中断和退出循环

JavaScript循环中断和退出循环

参考链接: https://juejin.im/entry/5884717a1b69e6005919f0d3

1. forEach

forEach 接收3个参数（当前项的值/当前项的索引/数组本身），它的返回值一直是 undefined， 所以forEach不能终止或跳出循环。
但是: 可以通过return true/false 退出当前循环(相当于continue)

注：如果需要终止或者退出循环可使用some(), every(); some() 是当内部return true时退出循环，every()是当内部return false退出循环。
MDN上还推荐 find(), findIndex() 以及通过filter过滤之后重新进行forEach

2. for

for 循环通过 continue退出当前循环，break跳出循环

3. map

map 也是不能中断或者跳出循环的
