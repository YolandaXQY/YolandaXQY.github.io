hexo搭建基本要点

起步

需要提前安装好 node npm

安装hexo

新建文章

新建页面

生成静态页面至 public 目录

开启预览访问端口 (默认端口4000)

hexo 发布(deploy) github

1. 在 _config.yml 中配置 git 信息
2. hexo-deployer-git
2. 通过 ssh 代理与 git 联系起来
1) 列出已存在的SSH Key
2) 生成密钥

以上这个命令会生成以 “your_email@example.com“ 为标签的ssh key , 然后连续两个enter就可以。

3）加入SSH Agent
4) 将SSH添加到Github中
5） 测试

hexo deploy 发布

域名设置

每个git账户只有一个域名可用，需要在你的仓库地址 setting 那边设置与git相同名字的域名
